# Refresh visual Rumo com zoom out

Este pacote aplica ao Controle de Qualidade de Via o mesmo padrão visual Rumo usado nos outros sistemas.

## O que foi alterado

- Paleta Rumo aplicada: azul institucional, azul digital, verde e amarelo.
- Logo e favicon Rumo adicionados em `assets/brand` e `public/assets/brand`.
- Tema claro como padrão, mantendo alternância para tema escuro.
- Dashboard, cards, filtros, tabelas, formulários, botões, gráficos e sidebar com escala reduzida.
- Altura dos gráficos reduzida para mostrar mais conteúdo na tela.
- Hero/topo redesenhado com painel institucional Rumo.
- Lógica de cálculo, localStorage, exportação, impressão, inspeção, HardScan e fissuras preservada.

## Publicação

Suba o conteúdo deste pacote no repositório. Após publicar no GitHub Pages, use `Ctrl + F5` no navegador para limpar cache.
